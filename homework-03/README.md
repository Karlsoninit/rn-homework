## Шаг 1
В этой части задания тебе нужно связать твою часть приложения с сервером
- Продолжаем вести работу в проекте который вы создали ранее ( домашнее задание 2)
- Установи Firebase в свой проект
- Зарегистрируй проект на [Firebase](https://firebase.google.com/) 
- Зайди внутрь проекта и поищи в вкладке ' настройки ' firebaseConfig в котором все данные о твоем проекте
  - не забудь настроить ' способ входа по email and password'

## Шаг 2
- В корне приложения создай файл config.js и поместим во внутрь firebaseConfig ( который ты нашел ранее )
- Отправляйся на официальную доку [Firebase](https://firebase.google.com/docs/reference/js/firebase.auth.Auth?authuser=0)
  где найдешь все функции для регистрации ( createUserWithEmailAndPassword, signInWithEmailAndPassword, onAuthStateChanged )
    - помни, все методы возвращают promise, а для обработки данных в нашем приложении мы используем async await

## Шаг 3 
- Ранее созданные формы нужно связать с firebase и его функциями
 - Для регистрации используй createUserWithEmailAndPassword
 - Для входа в уже созданным пользователем используй signInWithEmailAndPassword
 - Не забывай о переключении главных экранов и передай внутрь функции useRouter() данные от функции onAuthStateChanged
Если приходят ошибки о слишком коротком пароле или о невозможности зарегистрировать пользователи, выводи их внутри формы, 
чтобы user мог это видеть
